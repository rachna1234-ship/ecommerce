
E-Commerce Store

A simple full-stack e-commerce web application built using React, Node.js, Express.js, and MongoDB.

Features

- User Registration & Login
- Product Listing
- Add to Cart
- Order Placement
- Responsive Design

Tech Stack

Frontend

- React.js
- Tailwind CSS

Backend

- Node.js
- Express.js

Database

- MongoDB

Installation

Frontend

npm install
npm run dev

Backend

cd server
npm install
npm start

Project Structure

- "src/" → Frontend source code
- "server/" → Backend source code
